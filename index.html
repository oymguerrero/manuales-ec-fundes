¿De qué trata este proyecto?

Los Estándares de Competencia (EC) son documentos normativos que describen las funciones que una persona debe ser capaz de desempeñar para ser certificada por CONOCER (Consejo Nacional de Normalización y Certificación de Competencias Laborales). Cada EC se compone de elementos, criterios de evaluación de desempeños, productos, conocimientos y actitudes, hábitos y valores.

Mi CompañIA desarrolla un conjunto de Estándares de Competencia sobre IA aplicada a MiPyMEs, alineados con las necesidades reales del ecosistema productivo mexicano. Este sitio publica por ahora dos manuales: el Manual Maestro y el Manual del Estándar A · Implementación.

Sobre los otros tres Estándares
El sistema CONOCER contempla tres estándares adicionales para IA en MiPyMEs (Productos y servicios, Marketing digital, Transformación digital). Por ahora este sitio publica únicamente el Manual Maestro y el del Estándar A; los demás están en desarrollo. Para saber qué estándar te corresponde, el capítulo 5 del Maestro incluye un cuestionario de orientación.

Cómo colaborar en este manual

Este sitio es un borrador en construcción. Está pensado para que el equipo de Mi CompañIA pueda revisar, comentar y editar el contenido directamente desde GitHub o desde Antigravity.

  1. Abre la carpeta del repositorio en Antigravity (o cualquier editor con soporte de Git).
  2. Modifica el archivo HTML del EC en el que estás trabajando.
  3. Haz commit con un mensaje claro (ej. EC1: añadir descripción del elemento 2).
  4. Sube los cambios con push para que el resto del equipo los vea.
Nota: Toda la información del manual proviene de los archivos fuente: EC1–4 COMPLETO.docx, las Tablas de especificaciones y Todos los conocimientos.xlsx. Si encuentras una inconsistencia entre el manual y la fuente, la fuente manda.
